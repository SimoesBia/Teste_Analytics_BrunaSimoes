# Teste_Analytics_BrunaSimoes

## Descrição do Projeto

Este projeto simula um dataset de vendas referente ao ano de 2023, realiza a limpeza dos dados, executa análises exploratórias e propõe consultas SQL para extração de insights.

## Estrutura do Projeto

- simulacao_e_limpeza.py
- analise_exploratoria.py
- consultas_sql.sql
- data_simulated.csv
- data_clean.csv
- relatorio_insights.md

## Como Executar

1. Instalar dependências:
pip install pandas matplotlib

2. Executar:
python simulacao_e_limpeza.py
python analise_exploratoria.py

## Suposições

- Os dados foram simulados aleatoriamente.
- Os preços foram definidos dentro de uma faixa realista.
- Não foi considerada sazonalidade real na geração.
- A consulta de junho foi adaptada para 2023, pois o dataset contempla apenas esse ano.

## Tecnologias Utilizadas

- Python
- Pandas
- Matplotlib
- SQL
